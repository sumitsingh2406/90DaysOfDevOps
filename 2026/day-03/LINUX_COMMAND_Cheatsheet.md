#Basic commands in linux

#1 mkdir folder_name = to create folder in the director.

#2 cd = to change directory.
#3 cd .. = to change previous directory.
#4 rmdir folder_name = to remove folder.
#5 rm -r folder_name = to remove recursive folder in the directory.
#6 ls = to list the present directory.
#7 pwd = to show current directory.
#8 ls -l = to show every detail of current directory like file type, file name, file permission, owner of file etc.
#9 touch filename.txt = to create any text file.
#10 echo context > filename.txt = to write some content in the file.
#11 cat filename.txt = to read the content of the file.
#12 cp source_file destination_file = to copy file from source to destination.
#13 mv source_file destination_file = to move file from source to destination.
#14 wc filename.txt = to count the number of lines, words and characters in the file.
#15 sort filename.txt = to sort the content of the file in alphabetical order.
#17 echo "context" | tee filename.txt = to write some content in the file and also display it on the terminal.
#18 head filename.txt = to display the first 10 lines of the file.
#19 diff file1.txt file2.txt = to compare two files and show the differences between them.
#20 find . -name "filename.txt" = to search for a file in the current directory and its subdirectories.

#NETWORKING COMMAND IN LINUX

#1 ifconfig = to display the network configuration of the system.
#2 ping google.com = to check the connectivity to google.com.
#3 netstat = to display the active network connections and listening ports.
#4 ping -c 4 google.com = to send 4 packets to google.com and check the connectivity.
#5 traceroute google.com = to display the route taken by packets to reach google.com.
